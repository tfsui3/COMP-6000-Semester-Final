# COMP-6000-Semester-Final

Introduction: 
Are you tired of using a complicated and difficult-to-use student information management system? Well, our project is designed to completely transform the way students, teachers, and administrators access and interact with student data. Our goal is to create a system that is easy to use for everyone involved and eliminates the frustration of dealing with complicated software. Say goodbye to inefficient systems and hello to a better way of managing student information with our user-friendly solution.

The original intention behind this project design is that although Canvas is powerful and comprehensive, I believe that there are some shortcomings in certain features, such as the handling of student leave requests and attendance scores. Also, it's overly functional and not precise enough.

Proposal:
This system functional requirements are generally divided into functions for administrator use, teacher use, and student use.
